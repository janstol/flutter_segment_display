* updated example
* added widget tests

* updated segment styles for sixteen-segment display
* updated HexSegmentStyle diagonals

## [0.1.0] - 2019-05-11
* initial release
* added **seven-segment** and **fourteen-segment** display
* added DefaultSegmentStyle, HexSegmentStyle and RectSegmentStyle
